# 📋 Lista de Exercícios — +Prati

Repositório com 21 exercícios práticos de JavaScript desenvolvidos durante o curso **+Prati**, cobrindo fundamentos de lógica de programação, estruturas de dados e manipulação de objetos.

## 🚀 Tecnologias

- **Node.js**
- **JavaScript**
- **readline-sync** — leitura de dados via terminal

## 📁 Estrutura

```
listadeexercicios-maisprati/
├── exercicio1.js
├── exercicio2.js
├── ...
└── exercicio21.js
```

## 📝 Conteúdo dos Exercícios

| Exercício | Descrição |
|-----------|-----------|
| 01 | Cálculo de fatorial com loop `for` ou `while` |
| 02 | Sequência de Fibonacci com loop `for` |
| 03 | Leitura e impressão de nomes em ordem inversa |
| 04 | Cadastro de pessoas e listagem de menores de idade |
| 05 | Folha de pagamento com dedução de INSS |
| 06 | Pesquisa de salários e número de filhos |
| 07 | Matriz identidade 7x7 |
| 08 | Contagem de elementos negativos em matriz |
| 09 | Cálculo de peso ideal por sexo |
| 10 | Objetos: acesso e adição de propriedades |
| 11 | Filtragem de propriedades array em objeto |
| 12 | Combinação de objetos com precedência |
| 13 | Contagem de ocorrências em array de strings |
| 14 | Sumarização de vendas por vendedor |
| 15–21 | Exercícios complementares de lógica e estruturas de dados |

## ⚙️ Como executar

**Pré-requisitos:** Node.js instalado.

1. Clone o repositório:
```bash
git clone https://github.com/imgabrielimartins/listadeexercicios-maisprati.git
```

2. Instale as dependências:
```bash
npm install
```

3. Execute um exercício:
```bash
node exercicio1.js
```
